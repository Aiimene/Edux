# How to Run the App

## Prerequisites

- Node.js 18+ and npm

## Installation

1. Install dependencies:
```bash
npm install
```

## Running the App

### Development Mode

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Production Mode

1. Build the app:
```bash
npm run build
```

2. Start the production server:
```bash
npm start
```
